# 变量的语法扩展

```js
<script>
    // 1. 更新变量
    var myname = '卡卡西';
    console.log(myname);
    myname = '小明';
    console.log(myname);
    // 2. 同时声明多个变量
    var age = 29,
        address = '上海',
        wage = 60000;
    // 3. 声明变量的特殊情况
    var sex;
    console.log(sex);       //声明但为赋值，结
    qq = 110;
    console.log(qq);        //可以使用，不推荐
    console.log(tel);       //不声明不赋值，会
</script>
```
